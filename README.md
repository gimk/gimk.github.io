## 🛠️ Tech Stack

This website is hosted on **GitHub Pages**, and built using a modern, minimalist stack.

### Core

- **Framework:** [Astro](https://astro.build/) (v5)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Content:** [MDX](https://mdxjs.com/) & [Astro Content Collections](https://docs.astro.build/en/guides/content-collections/)

### Libraries & Tools

- **Typography:** [Fontsource](https://fontsource.org/) (DM Sans, Fraunces, JetBrains Mono)
- **Performance:** [Sharp](https://sharp.pixelplumbing.com/) for image optimization
- **Utilities:** `clsx`, `tailwind-merge`
- **Linting:** [ESLint](https://eslint.org/)
