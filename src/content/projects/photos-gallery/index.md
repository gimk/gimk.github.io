You can visit the live website here: [photos.pantoine.com](https://photos.pantoine.com)

I wanted to present my photos, which are currently hosted on Unsplash, in a more personal, gallery-like manner. The goal was to give time to the user to see and appreciate each photograph one by one, completely removing any visual noise from the interface.

To make each canvas feel more realistic and deeply integrated, I used the accent color of each photo as a subtle background element. Additionally, I decided to deliberately delay the appearance of the info board. This design choice ensures that the user focuses solely on the image first, before any text or context becomes visible.

## The Technical Side

For this project, I used my knowledge of Astro and GitHub Pages, which I had recently acquired from working on my portfolio.

However, the real accelerator was a heavy usage of **Vibe Coding**. By leveraging the Gemini CLI and Antigravity, alongside the Unsplash API, I was able to make my vision a reality incredibly fast. It took me only a few days—working just a few hours after work—to go from the initial idea straight to a production-ready website.

## Next Steps
